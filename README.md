# OpaqueVault Scoop Bucket

Scoop manifests for the [OpaqueVault CLI](https://opaquevault.com).

## Install

```powershell
scoop bucket add opaquevault https://github.com/huntrock17/scoop-bucket
scoop install ov
```

## Update

```powershell
scoop update ov
```

## Uninstall

```powershell
scoop uninstall ov
scoop bucket rm opaquevault
```

## About

This bucket is auto-updated by [GoReleaser](https://goreleaser.com) on every OpaqueVault release. The `bucket/ov.json` manifest is regenerated with each new version tag.

Source: https://github.com/huntrock17/opaquevault
